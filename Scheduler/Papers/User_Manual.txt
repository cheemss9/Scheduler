After opening the mobile application, the home screen will display it contains a button for setting an alarm and adding a schedule. It also displays a scroll bar that consists of days in the week. If you click Set alarm, it will display a text input that will allow you to add a certain alarm that you want then click "Set Alarm" to set the alarm. If you click Add Schedule, it will display a text input that will allow you to add your desired schedule for any day then click "Set Schedule" to add the schedule in the list. The user can also delete/update a schedule or an alarm by clicking the icon that you want to change.
